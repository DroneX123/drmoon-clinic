# Design System: Drmoon Clinic

> **VIBE:** Luxury, Medical, Elegant, Dark Mode.
> **LOCATION:** Hydra, Algiers.

## 1. Colors
- **Background:** `bg-slate-900` (Deep dark grey/black).
- **Text:** `text-slate-200` (Soft white).
- **Accent (Gold):** `text-yellow-500` or `bg-yellow-600` (Use for Buttons & Logos).

## 2. Typography
- **Headings:** Use 'Playfair Display' (Serif) for "Drmoon" and Headlines.
- **Body:** Use 'Lato' or 'Inter' (Sans-serif) for readable text.

## 3. Key Elements
- **Logo:** Use the `Moon` icon from `lucide-react` + text "Drmoon".
- **Buttons:** Rounded full (`rounded-full`). Gold background, Black text.
